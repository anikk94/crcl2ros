Detailed Description

CrclStatus is a class that encapsulates all the CRCL information. All lot of the knowledge is converting ROS oriented RCS data into codesynthesis Crcl representation and vice versa. CrclStatus maintains the unit a crcl session uses to transmit robot commands, latest robot status, etc.
