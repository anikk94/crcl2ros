Detailed Description

template<typename T, typename Comparator>
class RCS::CPriorityMessageQueue< T, Comparator >

The CMessageQueue offers a mutexed front to a STL/std deque. The queue is a LIFO data structure. Useful for safely sharing data between multiple threads.
