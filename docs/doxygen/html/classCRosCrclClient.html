Detailed Description

The CRosCrclClient class handles. At this point no ros cmd/status topics are used to communiate with ROS. INstead model information from gazebo is used. Possible in the future that real robot control would communicate through ROS cmd/status controller. topic subscription for CRCL status and advertising for CRCL commands.
